# User Management System

## Overview

This project is a simple user management system built using PHP with Object-Oriented Programming. It shows how different OOP concepts work together in a practical example.

---

## Project Structure

```
/user-management/
│── /App/
│   ├── /Core/
│   │   ├── AbstractUser.php
│   │   ├── AuthInterface.php
│   │   ├── LoggerTrait.php
│   │
│   ├── /Models/
│   │   ├── Admin.php
│   │   ├── RegularUser.php
│   │
│   ├── /Services/
│   │   ├── AuthService.php
│
│── index.php
│── autoload.php
```

---

## What this project does

* Creates two types of users: Admin and Regular User
* Allows login and logout
* Uses password hashing for security
* Logs activity for admin users

---

## OOP concepts used

* Abstract class
* Interface
* Trait
* Inheritance

---

## How to run

1. Put the folder inside `htdocs` (if using XAMPP)
2. Start Apache
3. Open:
   http://localhost/user-management/indexx.php
